# Blog Post Images

## How to Add Images

1. **Image Naming Convention:**
   - Name your image files exactly the same as the blog post slug
   - Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
   - Example: For a post with slug `mastering-subject-verb-agreement`, name the image:
     - `mastering-subject-verb-agreement.jpg` OR
     - `mastering-subject-verb-agreement.png` OR
     - `mastering-subject-verb-agreement.webp`

2. **Adding an Image:**
   - Simply place your image in this folder (`public/images/blog/`)
   - The system will automatically detect and use it
   - If no local image is found, it will use the external URL specified in the blog post

3. **Image Guidelines:**
   - Recommended size: 1200x630px (for optimal social media sharing)
   - Keep file size under 500KB for fast loading
   - Use descriptive, relevant images
   - Ensure images are properly licensed or owned by you

4. **Fallback Behavior:**
   - If you don't provide a local image, the system uses the `image` property from `blogPosts.tsx`
   - External URLs will still work as fallback
